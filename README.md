# SimpleASTM

A Simple ASTM over TCP Medical device simulator

## Quick Start

1. **Read CLAUDE.md first** - Contains essential rules for Claude Code
2. Follow the pre-task compliance checklist before starting any work
3. Use proper module structure under `src/main/swift/`
4. Commit after every completed task

## Development Guidelines

- **Always search first** before creating new files
- **Extend existing** functionality rather than duplicating  
- **Use Task agents** for operations >30 seconds
- **Single source of truth** for all functionality
- **SwiftUI structure** - organized for iOS/macOS development
- **Scalable** - start simple, grow as needed
- **Medical device simulator** - ASTM protocol implementation over TCP

## ASTM Protocol

This simulator implements the ASTM E1381 standard for transferring clinical laboratory data between instruments and computer systems over TCP connections.
